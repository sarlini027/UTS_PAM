# NAMA APLIKASI
Kalender To Do List

# DESKRIPSI APLIKASI
Aplikasi ini menggunakan API kalender. Dalam aplikasi ini user dapat melihat kalender formal sekaligus mencatat kegiatan pada hari tertentu dan dapat diatur alarm ketika kegiatan telah selesai.
